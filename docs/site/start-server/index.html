    创建标识服务器¶

    OAuthApp是一个基于AspNetCoreIdentity、IdentityServer4的框架，可以帮助快速搭建具有开放平台的用户中心。

    参考如下步骤，搭建标识服务器。请确保您已经安装了Visual Studio 2019、DotNet Core 3.1、SQL Server。

    1，创建应用程序¶

    2，安装Nuget包¶

    对项目的依赖项点右键，选择管理Nuget程序包

    输入OAuthApp，然后点击安装

    3，配置¶

    3.1 appsetting.json¶

    对项目依次点击鼠标右键——属性——调试，复制项目网址.

    复制下面代码到 appsetting.json 文件，注意将Host节点替换为当前项目网址，结尾不要带“/” 。

    DataBaseConnection 可替换为实际数据库的地址，SMS 和 Email 使用的是SendCloud的服务。

    "Host": "当前项目网址，结尾不要到/",

      "apiUser": "可空",

    "Email": {
      "apiUser": "可空",

      "fromEmail": "可空",
      "fromName": "可空"
    }
  }

    4， 启动¶

    启动项目并预览。默认用户账号：admin@admin.com，密码：123456aA!，拥有所有权限。

    5， Swagger UI¶

    在浏览器中打开 {项目网址} /swagger/index.html可访问API文档。

    可访问 {项目网址} /grants，撤销对指定client的授权。

    6，Postman¶

    使用 Postman 测试现有接口。
